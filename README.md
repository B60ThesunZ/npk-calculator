# NPK Calculator (เครื่องคำนวณสูตรปุ๋ย)

เครื่องมือคำนวณสูตรปุ๋ยผสม N-P-K จากแม่ปุ๋ย

## แม่ปุ๋ยที่ใช้
- 46-0-0
- 18-46-0
- 0-0-60

## วิธีใช้
1. ใส่เปอร์เซ็นต์ N-P-K ที่ต้องการ
2. ใส่น้ำหนักรวมที่ต้องการ (กิโลกรัม)
3. กดคำนวณ

## Features
- คำนวณปริมาณแม่ปุ๋ยแต่ละชนิดที่ต้องใช้
- แสดงผลในหน่วยกิโลกรัม
- คำนวณปริมาณ Filler ที่ต้องเพิ่ม

## Requirements
Python packages ที่จำเป็น:
```
streamlit==1.50.0
```

## Run Locally
```bash
streamlit run NPK.py
```
